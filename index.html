FlashCards sobre programação

feito por Pedro R.O.
O que é uma variável em programação?
Uma variável é um espaço de memória nomeado usado para armazenar dados que podem ser manipulados durante a execução de um programa.
Qual é a diferença entre um comentário de uma linha e um comentário de múltiplas linhas?
Um comentário de uma linha é usado para explicar uma única linha de código. Um comentário de múltiplas linhas abrange várias linhas
O que é uma função em programação?
Uma função é um bloco de código reutilizável que executa uma tarefa específica quando chamado. Ela pode receber parâmetros e retornar valores.

O que é uma estrutura de repetição (loop)?
Uma estrutura de repetição permite executar um bloco de código várias vezes.
Explique a diferença entre uma variável global e uma variável local.
Uma variável global é declarada fora de qualquer função e pode ser acessada em qualquer parte do programa. Uma variável local é declarada dentro de uma função e só pode ser usada dentro dela.
O que é recursão e quando ela é usada?
Recursão é quando uma função chama a si mesma para resolver um problema, dividindo-o em subproblemas menores. É usada em problemas como cálculo de fatorial ou percorrer árvores.

O que é complexidade de tempo e como ela é analisada usando a notação Big-O
Complexidade de tempo mede o tempo que um algoritmo leva para executar em relação ao tamanho da entrada. A notação Big-O descreve o pior caso de crescimento assintótico. Exemplos:
O(1): Tempo constante (ex.: acessar um elemento de um array).
O(n): Linear (ex.: percorrer uma lista).
Explique o conceito de programação orientada a objetos (POO)
POO é um paradigma que organiza o código em objetos, que combinam dados (atributos) e comportamentos (métodos).
O que é concorrência e como ela difere de paralelismo em programação?
Concorrência é a capacidade de um programa gerenciar múltiplas tarefas que podem ser executadas em ordem não determinística, compartilhando recursos (ex.: threads alternando em uma CPU). Paralelismo é a execução simultânea de tarefas em múltiplos processadores ou núcleos.
